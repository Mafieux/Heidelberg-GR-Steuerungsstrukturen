# Heidelberg-GR-Steuerungsstrukturen

Repository zur Erarbeitung von **Steuerungsstrukturen für den Heidelberger Gemeinderat** im Kontext des städtischen Haushalts.

## Zweck

Dieses Repo dient als Ablage und Versionierung von Arbeitsergebnissen rund um:

- Steuerungslogik und -instrumente des Heidelberger Gemeinderats
- Haushaltsstruktur und mehrjährige Datenbasis
- Hypothesen zu Wirkungshebeln
- Vorbereitungen und Outputs für Sitzungen / Termine

## Status

Initiales Repo — Inhalte werden schrittweise ergänzt.

## Hinweis zur Ordnerstruktur

Das lokale Arbeitsverzeichnis enthält zusätzliche, nicht versionierte Arbeitsdateien (siehe `.gitignore`). Nur explizit fürs Repo freigegebene Dateien werden hier eingecheckt.
